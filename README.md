**Гайд по использованию репозитория

***Установка репозитория с кодом на собственный пк
>1. Открыть терминал в среде разработки, которую ты используешь. Если это VS Code - комбинация клавиш ctrl + ~ или ctrl + ё (клавиши одни и те же)
>2. В терминале вписать команду git clone https://github.com/whunotexplain/klim_kp
>3. Затем у тебя в директории появится сама директория репозитория. Либо ты заходишь в нее и пользуешься кодом, если ты хочешь так, то введи в терминал эту команду ```cd klim_cp_code```. Если тебе неудобно использовать перемещение через терминал, то в проводнике вынеси директорию с кодом туда куда удобно и просто в нее перейди потом.

**База данных

__При написании и тестинге проекта я использовал postgresql и в целом код написан с использованием этой СУБД.__

**В файле тебе гадо будет делать исправления для подключения к собственной СУБД

**Детальный гайд:

>1. Код с подключением к базе данных находится в Директории: database в файле connection.py

>2. Открой файл и найди: 
```
class Database:
    def __init__(self):
        # Инициализация атрибутов
        self._processed_files = {}
        
        try:
            self.conn = psycopg2.connect(
                host='localhost',  #Это остается неизменным
                port='5432', #Это остается неизменным
                dbname='resume_db', #Замени на название базы данных, которую ты будешь использовать
                user='postgres', #Замени на имя юзера базы данных, которой ты создал
                password='12345', #Замени на пароль базы данных, которой ты создал
                cursor_factory=RealDictCursor,
                client_encoding='UTF-8'
            )
            print("✓ Database connected successfully")
            self.create_tables()
        except Exception as e:
            print(f"✗ Database connection error: {e}")
            print("Приложение будет работать без базы данных")
            self.conn = None
```

>3. Сохрание изменения в файле ctrl + S или в ручную через окно IDE
